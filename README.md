## TaskManagement

واجهة إدارة مهام (Kanban) مبنية باستخدام **React**, **TypeScript** و **Tailwind CSS** مع دعم:

- **لوحة مهام** بنمط كانبان مع أعمدة الحالة (للتنفيذ، قيد التنفيذ، مكتملة، متوقفة).
- **نافذة منبثقة** لإضافة مهمة جديدة مع التحقق من صحة الحقول.
- **سحب وإفلات** للبطاقات بين الأعمدة وتحديث الحالة.

### التقنية المستخدمة

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**

### الأوامر الأساسية

بعد تثبيت الحزم (`npm install`):

```bash
npm run dev     # تشغيل بيئة التطوير
npm run build   # إنشاء نسخة الإنتاج
npm run preview # معاينة نسخة الإنتاج
```
